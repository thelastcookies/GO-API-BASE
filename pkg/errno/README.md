# errno

公共系统错误码。

## 错误代码说明

| 1      | 00     | 02     |
|:-------|:-------|:-------|
| 错误级别代码 | 服务模块代码 | 具体错误代码 |

- 服务级别错误：
    - 1 为系统级错误；
    - 2 为服务级错误，通常是由用户非法操作引起，返回业务错误码。
- `code = 0` 说明是正确返回，`code > 0` 说明是错误返回
- 上述说明同样适用于业务错误码。
